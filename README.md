# GeoIP cmd


    geoip 217.30.184.170
    IP: 217.30.184.170
    Reverse: vhost.kapsi.fi (OK)
    Country(FI): Finland
    Region: None
    City: None
    LatLon: 64.0, 26.0


## Install

Copy bin/geoip to one of the directories in your $PATH.

Fetch / update geoip database.

     ./bin/update_geoip.sh
